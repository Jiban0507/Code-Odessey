## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Jiban0507/Mini-Project-Collection-Page.git
cd Mini-Project-Collection-Page
```

### 2. Run the project

No setup needed — just open:

```bash
index.html
```

> **Tip:** Use VS Code Live Server for better development experience.
